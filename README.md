## LoLAccountChecker

* This tool will give you some information about your League of Legends accounts and store them in a file.

### Requirements:
 * Windows 7 or later
 * [.NET Framework 4.5](https://www.microsoft.com/en-us/download/details.aspx?id=30653) or later
